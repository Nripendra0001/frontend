All Required Certificates

  • Graduation Certificate (if required)
  • Aadhar Card / Voter ID
  • Caste Certificate
  • Domicile Certificate
  • EWS Certificate
  • Disability Certificate (if any)
  • Ex-serviceman Certificate
  • Character Certificate
  • Passport size photos

Photo & Signature Rules

  • White/light background photo
  • No cap, goggles, filters
  • Recent photo (last 3‚Äì6 months)
  • Signature black/blue pen
  • JPG/JPEG format only
  • Usually 20kb ‚Äì 200kb size
  • No blur or shadow
  • Full face visible

‚ö† Common Form Mistakes

  • Wrong name spelling
  • DOB mismatch with marksheet
  • Wrong category selection
  • Invalid photo/sign upload
  • Wrong mobile/email
  • Incomplete form
  • Wrong exam center
  • Payment not confirmed

Physical & Medical Rules

  • Height & chest standards
  • Weight & BMI check
  • Running test rules
  • Long jump / high jump
  • Eye sight standards
  • Color blindness test
  • Flat foot / knock knee
  • Tattoo rules

Verification Process

  • Original documents required
  • Multiple photocopies
  • Photo ID mandatory
  • Self-attested copies
  • Biometric verification
  • Category & address check
  • Education proof check
  • Final merit confirmation

‚ùå Rejection Reasons

  • Multiple forms
